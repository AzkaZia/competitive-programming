# cf-journey

Tracking my climb from Codeforces 980 → 1800, starting June 2026.

Goal: get strong enough at competitive programming to find solid ICPC teammates going into freshman year at LUMS (O-Week: August 24, 2026).

## Structure

Problems are organized by **topic/tag**, not by date. Each file is named after the problem ID (e.g. `1023B.cpp`). Daily logs and progress tracking live in a separate spreadsheet — this repo is purely solution code.

```
cf-journey/
├── binary-search/
├── two-pointers/
├── greedy/
├── math/
├── implementation/
├── sortings/
├── strings/
├── brute-force/
├── number-theory/
├── dp/
├── data-structures/
├── graphs/
└── games/
```

## File format

Every solution file starts with a short header so I (or anyone else) can understand the approach without re-reading the whole problem:

```cpp
// CF 1023B - [Problem Name]
// Approach: one-line description of the core idea
// Time: O(n log n)
```
